## ğŸ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas URI
- AWS Credentials (for deployment)

### 1. Clone & Install

```bash
git clone <repository-url>
cd daily-spend
```

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
The app will run at `http://localhost:3000`.

### 3. Backend Setup

```bash
cd backend
npm install
# Configure .env with MONGO_URI and JWT_SECRET
npm start
```
For local development, the backend runs on port 5000.
